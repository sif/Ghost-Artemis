Services

Reconnaissance, Footprinting & Passive Scanning

  • Inventory of external infrastructure
  • Topological map of the network
  • Locating traffic routes, physical location, domain registry
  • Identifying IPv6 use, IP block usage, ISP
  • Scanning all 65536 ports with varied scans like XMAS, SYN, FIN, NULL, etc

Active Port Scans, Fingerprinting & Service Probing

  • Open/closed/half/filtered port lists
  • Firewall on gateway interfaces, TCP sequence number prediction, IPID prediction
  • Identifying hosted service on standard and non-standard ports
  • Detecting service running on the open ports

Password Service Strength Testing

  • Brute-forcing logins of known services on standard or non-standard ports with custom password list of 5000 passwords
  • Testing default passwords for known services

About Us

Ghost Artemis went from being a toy to a project to becoming an small organization. With electronics often being a central part of our lives, we want to build a platform for a better future.

I want to leave behind a better world. If I were to be reborn, I would want to have built a tree that provides the shade. Cybersecurity (and ultimately computer science) will pave that path for us.

Security does so by allowing us to live in peace and comfort. It allows us to think and live in dignity. We can only do this when we are allowed to be different. Come build this world with me!
